# Mark Report Finalizer

A desktop application built with **JavaFX (OOP principles)** and **SQLite** to help teachers easily generate student rankings and well-structured mark sheets — especially useful for those who may not be comfortable working with Excel.

---

## 🚀 Features
- Generate ranking and marking sheets quickly.
- Simple and user-friendly GUI with JavaFX.
- Portable installer for easy setup (no manual Java installation required).
- Data stored securely with SQLite.

---

## 🛠️ Technologies Used
- **JavaFX** (UI + OOP principles)
- **SQLite** (lightweight database)
- **Launch4j** (packaging JAR into EXE)
- **Inno Setup** (installer creation)
